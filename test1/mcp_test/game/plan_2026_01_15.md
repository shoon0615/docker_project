# Game Development Plan (2026-01-15)

This is a revised, more focused plan for the 2D auto-shooter game.

## Technology Stack
*   **Language:** JavaScript (ES6+)
*   **Platform:** Web Browser
*   **Core API:** HTML5 Canvas
*   **Libraries:** None (pure JavaScript)

## File Structure
*   `index.html`: Main HTML file, contains the `<canvas>` element.
*   `style.css`: CSS for basic page and canvas styling.
*   `game.js`: Core game logic, including player, enemies, systems, and rendering.
*   `plan_2026_01_15.md`: This planning document.
*   `plan_2026_01_15_ko.md`: Korean version of this planning document.

## Phase 1: Foundational Features (Partially Complete)

*   [x] Basic player movement (WASD/Arrows).
*   [x] Basic enemy spawning and movement.
*   [x] Basic automatic shooting.
*   [x] Basic collision detection.
*   [x] Basic score and game over screen.

## Phase 2: Core Gameplay Mechanics (Completed)

*   [x] **Experience and Leveling:**
    *   [x] Enemies drop experience gems upon death.
    *   [x] Player collects gems to gain experience.
    *   [x] Implement a player level that increases with experience.
    *   [x] On level up, slightly increase a player stat (e.g., shooting speed or player speed).
*   [x] **Special Move:**
    *   [x] Implement a screen-clearing "bomb" attack activated by a mouse click.
    *   [x] This special move will have a cooldown period.
*   [x] **UI Elements:**
    *   [x] Add a UI bar to show current experience and progress to the next level.
    *   [x] Display the current player level on the screen.
    *   [x] Add a UI element to show the cooldown status of the special move.

## Phase 3: Refinement (Completed)

*   [x] Refine enemy spawn rates and player stat progression for better game balance.
*   [x] Polish the simple UI elements for clarity.
